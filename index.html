Monster In a Box

Skills: Arduino, C++, Basic Electricity

This is my version of a Monster-in-a-Box Holloween prop. It is driven by an Arduino microcontroller and powered by 12v and 5v power supplies. It uses a windshield wiper motor to turn a homemade 'sprocket' that lifts the lid about 1/2" before loudly dropping it back down. The chains rattle and provide additional sound/scare factor.

The LED strip displays a random sequence programmed to give the appearance of fire inside the box. Once the motor is activated by the IR sensor, the light sequence changes to full red and the motor is engaged. Awakening the creature within!!

Color Organ

Skills: Schematic Reading, Soldering, Linear Devices

A Color Organ is a reactive light instrument that filters different frequencies from a sound wave. These are my attempts. The first is a smaller version and both were from schematics I found online. The prototype was done on a breadboard and then a permanent one was made. This project is limited from a coding aspect but the learning value was massive.

Version two was done on a bigger board and I used the layout from an online example. I've thought about using this board to learn CAD and design my own 3D printed case. As it is now though, the device works great and uses very little power. It doesn't have any external speakers but it can easily be fed with an audio splitter cable.
